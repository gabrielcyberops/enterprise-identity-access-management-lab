# Enterprise Identity & Access Management Lab

## Descripción

Enterprise Identity & Access Management Lab es un laboratorio práctico enfocado en Active Directory, Identity & Access Management (IAM), administración de infraestructura Windows y aplicación de controles de seguridad utilizados en entornos empresariales.

El proyecto busca desarrollar conocimientos sólidos sobre administración de identidades, delegación de privilegios, auditoría, recuperación ante incidentes y hardening de Active Directory mediante una metodología progresiva basada en escenarios reales.

Cada módulo sigue un proceso estructurado de diseño, implementación, validación y documentación, priorizando la comprensión técnica y la capacidad de explicar cada decisión desde una perspectiva profesional.

---

## Objetivos

* Desarrollar habilidades prácticas de administración de Active Directory.
* Comprender los fundamentos de Identity & Access Management (IAM).
* Implementar modelos seguros de delegación de privilegios.
* Aprender procedimientos de Backup y Recovery aplicados a Active Directory.
* Mejorar capacidades de Auditing y Monitoring.
* Aplicar controles de Hardening para reducir riesgos de seguridad.
* Documentar todas las implementaciones de forma profesional y reproducible.
